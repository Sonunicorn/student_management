\# 🎓 Student Management System



\## 📌 Project Overview

This is a simple Student Management System built using \*\*Java, JDBC, MySQL, HTML, and CSS\*\*.  

It allows users to perform CRUD operations on student records.



\---



\## 🚀 Features

\- Add new student

\- View student details

\- Update student information

\- Delete student records

\- Database connectivity using JDBC



\---



\## 🛠️ Tech Stack

\- Java

\- JDBC

\- MySQL

\- HTML

\- CSS



\---



\## 🗄️ Database

\- MySQL is used as backend database

\- JDBC is used for connectivity



\---



\## ▶️ How to Run

1\. Import project into Eclipse/IntelliJ

2\. Setup MySQL database

3\. Update DB credentials in `DBconnection.java`

4\. Run `Main.java`
